# animekiosk
It's an Electron-based desktop kiosk browser built for anime streaming. Here's what it does:  🔒 Locked browser — Only allows whitelisted anime sites (animepahe.pw, kwik.si, kwik.cx, pahe.win) 🚫 Ad-free — Aggressively blocks ads through 4 layers (network interception, navigation events, popup blocking, and a JS guard injected into pages)
